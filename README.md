# time_preferences_in_forecasting
This repository contains selective code and data for the analysis from the paper "Time preferences in forecasting". 

The main analysis can be found in the file "empirical analysis."
The robustness check is located in "robustness check".
The "topics" file includes the list of topics along with labels indicating whether each event is time-varying or time-fixed.
Combined with the individual Metaculus predictions, these files provide all the material necessary to replicate the study.
If you have questions or need assistance replicating the analysis, please reach out to the email address provided in the scripts.
